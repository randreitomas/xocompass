XoCompass is an enterprise-grade B2B analytics dashboard built with React, Vite, TypeScript, and Tailwind CSS.

It provides a multi-view experience:

- Simplified business metrics view
- Advanced technical metrics view
- Time Series Model Lab
- FAQs and help content

## Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- React Router
- `lucide-react` for icons
- `recharts` for charts

## Getting Started

1. Install dependencies:

```bash
cd xocompass
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the app in your browser at the URL shown in the terminal (typically `http://localhost:5173`).

The default route loads the **Simplified Metrics** dashboard with the main layout (sidebar + header).
